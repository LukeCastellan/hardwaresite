Computer Case

The Computer Case is essentially a large box which is used to hold all the hardware components (Eg. CPU, RAM etc.) together and protect them from dust.

Computer Motherboard

The Motherboard is the most crucial component required to make a computer function, it is a Printed Circuit Board (The largest in the computer, in fact.) that holds components like the CPU and GPU while also communicating with other components such as the HDD and Audio Card in order to perform calculations, file read/write etc. It also receives electrical power from the PSU and allocates it to components accordingly.

Riser Card

The Riser Card is a Printed Circuit Board that allows for the addition of Expansion Cards when the case cannot offer enough vertical space, the card plugs into the motherboard and allows the Expansion Cards to be installed horizontally, saving space. It also picks up a multitude of signal lines (often bused) via a single connector (usually an edge connector) on a motherboard and distributes them via dedicated connectors on the card.

Power Supply Unit

The Power Supply Unit is a converter to switch AC Power to low voltage DC power that components are able to use. Some PSUs are manually switched for the input voltage, while others automatically adapt based on the AC source.

Random Access Memory

The Random Access Memory is a type of volatile memory (Will lose contents without electrical power.) designed as temporary data storage for when the CPU is processing. It cannot be used for long term data storage as it is easily overwritten and will lose all contents without electrical power.

Central Processing Unit

The Central Processing Unit is what allows for computer itself to run. It uses billions of microtransistors and other extremely small components to perform arithmetic operations, data input/output and control operations given by binary instructions. In terms of larger components, the CPU is made of Control Units (To interpret binary instructions, and Arithmetic Logic Units (to perform calculations).

CPU Fan & Heatsink

The Heatsink and Fan assembly allows the CPU to maintain reasonable temperature while operating. This is because the CPU performs many calculations per second, while being extremely small in volume. Thus creating a lot of heat it cannot disperse itself. That is where the assembly comes in to use. The Heatsink is essentially a large block of metal/a heat exchanger that absorbs heat from the CPU using Thermal Paste; While the fan disperses any remaining heat

Hard Disk Drive(s)

The Hard Disk Drive is a magnetic data storage device, using a magnetic platter as the storage itself and magnetic heads used with acutator arms in order to read/write data to its surface. It is a non-volatile memory (Data can be kept without power)

How the Components Interact Together

When the user wants to perform a task on the computer, they will run a program to do that. In order for the program to process data and perform necessary computations, there are multiple steps to that. First, the program sends instructions to the motherboard, then, the data required is loaded into the RAM from the HDD for faster access and read/write during processing. After those steps, the CPU will receive instructions from the motherboard to perform calculations, and the GPU could also receive instructions to render polygons/objects if the program requires it to. All of those calculations are kept in sync with the use of the internal clock, to ensure the process can complete all at once.
